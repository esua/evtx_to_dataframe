# evtx_to_dataframe
Convert Windows EVTX event log file to Pandas DataFrame.


# Requirement
* python-evtx 0.6.1
* pandas 1.0.1
* xmltodict 0.12.0


# Installation
```sh
pip install python-evtx
pip install pandas
pip install xmltodict
```


# Usage
```sh
python evtx_to_dataframe.py filename.evtx
```
